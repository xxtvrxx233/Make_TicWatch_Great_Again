# 让 TicWatch 再次伟大
该项目包含多个 Magisk 模块，用于优化运行 Wear OS 3.5 的 TicWatch 设备。

您需要安装 [Kitsune Mask](https://github.com/HuskyDG/magisk-files/releases) 27.0+ 才能安装这些模块。

# 兼容的设备
-TicWatch Pro 5

-TicWatch Pro 3

-TicWatch Pro 3 Ultra

(TicWatch Pro X)

-TicWatch E3

_注意:您必须已经通过某些手段更新到Wear OS 3.5才能使用这些模块_
